Doctype

La declaración doctype

En un documento HTML, por norma general, se le tiene que indicar al navegador que version de HTML vamos a implementar. De este modo, el navegador sabrá como implementar las etiquetas dentro del archivo

En HTML5 se le indica mediante el atributo "html":

<!Doctype html>

A modo de referencia, añadiré que versiones anteriores se declaraba de la siguiente manera:

Para HTML4

<!Doctype HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

Para XHTML 1.1

<!Doctype html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">

Cabe añadir que el elemento !Doctype no hace distinción entre mayúsculas o minúsculas
