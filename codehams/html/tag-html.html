Elemento HTML

    HTML

    El elemento HTML define el contexto principal y representa la raiz de todo documento HTML. Este elemento puede albergar en su interior todos los demás elementos HTML excepto la declaración <!Doctype>

    Siempre debería de ir acompañado del atributo lang para especificar en qué idioma se esta desplegando la página

    <html lang="es">

    Otro atributo que se le puede añadir es xmls con el valor "http://www.w3.org/1999/xhtml" en el caso de que se necesite representar elementos XHTML

    En tal caso, también debería ser necesario indicar el idioma de los elementos XML con el atributo xml:lang

    <html lang="es" xml:lang="es" xmls"http://www.w3.org/1999/xhtml">

    El elemento <html> soporta todos los atributos globales del estandar HTML5

    ¿Cómo hago para colocar contenido en otro idioma al declarado?

    Bueno.. en este caso se podría usar el lang en el lugar donde se necesita usar otro idioma con el elemento <span> para especificar el lenguaje

    <p>"Tengo un gato de color azul" en Japonés: <span lang="ja">私は青い猫を飼っています</span></p>

    Esto daría como resultado:
    "Tengo un gato de color azul" en Japonés: 私は青い猫を飼っています
