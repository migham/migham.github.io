Elemento <head>

    ¿Qué contiene la cabecera?

    Pues la mía nada.. =(

    Pero el elemento <head> de un documento HTML contiene toda la metadata sobre el HTML en cuestión y es la parte que no se mostrará. Es donde se añade el título de la página, estilos y enlaces por ejemplo para personalizar el favicon o scripts...

    Por ejemplo:

    <!Doctype html>
    <html lang="es">
    <head>
    <title>Titulo</title>
    </head>
    <body>
    <!-- Contenido de la web -->
    </body>
    </html>

    Otro ejemplo sería añadiendo el set de caracteres para UTF-8:

    <!Doctype html>
    <html lang="es">
    <head>
    <meta charset="utf-8" />
    <title>Titulo</title>
    </head>
    <body>
    <!-- Contenido de la web -->
    </body>
    </html>

    Notese que el elemento <meta /> no poseé una etiqueta de cierre, sino que en el mismo elemento se incluye el cierre con el slash ó barra inclinada.

    Otro ejemplo añadiendo estilos:

    <!Doctype html>
    <html lang="es">
    <head>
    <meta charset="utf-8"/>
    <title>Titulo</title>
    <style>
    h1{ color: red;}
    p{ color: blue;}
    </style>
    </head>
    <body>
    <h1>Titulo</h1>
    <p>Esto es un párrafo</p>
    </body>
    </html>

    Atributos

    El elemento <head> admite todos los atributos globales

    • <title>
    • <style>
    • <base>
    • <link>
    • <meta>
    • <script>
    • <noscript>
